# ICU 输液泵计算器

纯前端 ICU 输液泵速度换算工具，支持剂量算泵速、泵速反算剂量、药物模板、自定义药物、本地 `localStorage` 保存和深色模式。

## 使用

打开 `index.html` 即可使用。部署到 GitHub Pages 后，可在手机浏览器中访问并添加到主屏幕。

## GitHub Pages

仓库根目录已经包含 `index.html`、`style.css`、`app.js`。在 GitHub 仓库中进入 `Settings -> Pages`，选择：

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

保存后等待 1-2 分钟，即可通过 GitHub Pages 访问。
